# JobBoard Website

## Description
This project is a job board platform built using HTML, CSS, and JavaScript. 
It allows users to browse job listings, filter jobs based on salary and work type, 
and apply through a dynamic application form.

## Features
- Job search by title or company
- Salary range filtering
- Remote and On-site job filters
- Dynamic job card generation
- Job application modal form
- Form validation

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open index.html in any web browser
